# 免费模式 / 无内购配置

> 位置：App Store Connect → Pricing and Availability / Monetization。
> 本应用完全免费，不需要创建任何 In-App Purchase。

## 结论

| 项目 | 配置 |
|------|------|
| App 价格 | 免费 |
| App 内购买 | 无 |
| 订阅 | 无 |
| 消耗型项目 | 无 |
| 非消耗型项目 | 无 |
| 付费墙 | 无 |
| 试用期 | 无 |
| 外部支付 | 无 |

## App Store Connect 操作

1. App 价格选择免费。
2. 不创建 In-App Purchases。
3. 不创建 Subscriptions。
4. 版本提交页面无需勾选任何内购项目。
5. 审核备注中说明 “The app is completely free. There are no in-app purchases, no subscriptions, no paid features, and no paywall.”

## 与代码一致性核对（2026-06-29）

- 未发现 StoreKit / Product / Transaction / purchase / restore 等内购代码引用。
- 未发现 `IAP`、`subscription`、`订阅`、`内购`、`购买` 相关业务逻辑。
- UI 中没有付费墙或解锁入口。
- App 功能均可在授权 HealthKit 后免费使用。
- 设置页“发现我的其他应用”仅打开开发者 App Store 页面，不影响 KingFit 的免费功能，不构成本 App 的内购或付费墙。

## 注意
