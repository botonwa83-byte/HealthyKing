# 年龄分级问卷（Age Rating）

> 位置：App Store Connect → App 信息 → 年龄分级 → 编辑。
> 本应用展示健康与运动指标，包含心率、血氧、睡眠、恢复评分、训练负荷和一般健康建议。建议保守结论：12+。

## 建议选择

| 问题项 | 建议答案 |
|--------|----------|
| 卡通或幻想暴力 | 无 |
| 现实暴力 | 无 |
| 露骨/恐怖暴力 | 无 |
| 亵渎或低俗幽默 | 无 |
| 成人/性暗示题材 | 无 |
| 裸露 | 无 |
| 色情内容 | 无 |
| 恐怖/惊悚题材 | 无 |
| 酒精、烟草或毒品的使用或相关内容 | 无 |
| 模拟赌博 | 无 |
| 医疗/治疗信息 | 偶尔/轻微 |
| 不受限制的网络访问 | 否 |
| 赌博与竞赛 | 无 |

## 为什么不建议 4+

本应用不是医疗软件，也不提供诊断、治疗或疾病预防功能；但它会展示健康数据和运动恢复建议，包括血氧、心率、睡眠、训练负荷等。为了健康类审核更稳妥，建议在“医疗/治疗信息”项选择“偶尔/轻微”，最终分级通常会落在 **12+**。

如果 App Store Connect 后台对“医疗/治疗信息”的描述与你看到的选项不同，原则是：

- 不选择“频繁/强烈”；
- 不把 App 描述成医疗诊断工具；
- 保持“普通健康与运动趋势参考”的定位；
- 最终年龄分级以 ASC 实际计算结果为准。

## 其它分级相关开关

- 是否含用户生成内容（UGC）：否。
- 是否为 Kids Category：否。
- 不受限制 Web 访问：否。
- 广告：无。
- 赌博：无。
- 竞赛：无。

## 预期结果

建议年龄分级：**12+**

## 审核口径

如审核询问，可说明：

```text
The app displays general wellness and fitness trend information based on HealthKit data. It is not a medical device and does not diagnose, treat, cure, or prevent any disease. The app includes a clear disclaimer during onboarding and in Settings.
```
